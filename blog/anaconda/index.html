Anaconda

Aug 3 2020

Introduction

Every programmer should know (or be familiar with) a couple of programming languages. Some languages like Java or Scala are good for production-ready applications. On the other hand, some languages like Python are good for scripting, tooling or creating proof of concepts. If you want to dig deep into being a pragmatic programmer I highly recommend reading The Pragmatic Programmer: From Journeyman to Master by Andrew Hunt and David Thomas.

Python

Although I’ve been working with JVM languages in my career, I use Python for various tasks. Python is advertised as a batteries included language but from time to time there is something missing and you have to download a package. Of course, you can install Python and Pip from your OS package manager, but still you have to download packages manually.

Anaconda

Fortunately there’s another snake that comes to the rescue of troubled programmers - Anaconda. It is mainly used by data scientists, but everybody can benefit from it. Anaconda contains many wonderful packages like numpy, pandas or scipy. Everything is ready for use and works great on every OS. Imagine having a JDK bundled with Guava, Spring or Apache with both CLI and GUI to manage it.

Conclusion

I hope that the next time instead of installing Python you will give Anaconda a chance. Or maybe you already switched from raw Python to Anaconda?
